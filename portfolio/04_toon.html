Project Toon Shading

3D Graphics

Project Overview

In the D04ToonShading demo I built a real time toon shading framework in OpenGL, showcasing three stylized lighting techniques Custom Range, Uniform Range, and Gradient Texture discretization augmented with adaptive anti aliasing and linear fog. I generated a procedural Trefoil Knot mesh, vertex & fragment shaders for Blinn Phong lighting, and provided an ImGui interface for live parameter control.

Tasks Completed

  • Procedural Geometry: Implemented create_trefoil in Mesh.cpp to generate and normalize a high resolution Trefoil Knot.
  • Vertex Shader (toon.vert): Computed model→view→projection transforms, passed view space normals via a 3×3 normal matrix, and forwarded view position for fog.
  • Custom-Range Toon (01_custom_range_toon.frag): Four user defined thresholds with smoothstep() or step() for anti-aliased banding.
  • Uniform-Range Toon (02_uniform_range_toon.frag): N equal diffuse chunks, discretized mathematically with optional derivative based smoothing.
  • Gradient-Texture Toon (03_gradient_texture_toon.frag): 1D lookup texture for artist driven shading, with optional multi sample anti aliasing around each texel.
  • Anti-Aliasing: Leveraged fwidth() and smoothstep() to soften both diffuse bands and specular edges adaptively.
  • Linear Fog: Applied a view space depth mix uniformly across all three toon modes for atmospheric depth cueing.
  • UI Integration: Exposed all parameters thresholds, chunk count, gradient selection, fog settings, light direction, shininess via ImGui for real-time tuning.
  • Cross Platform Delivery: Used GL:: wrappers, GLM, SDL+ImGui, clang-format, and verified builds on Windows x64 and webgl.

Reflection

Balancing sharp, stylized bands with smooth visual transitions was the core challenge. Implementing derivative based anti aliasing taught me how fwidth() adapts to screen space variation and maintains crisp edges at any zoom. Correctly computing and passing the normal matrix reinforced my understanding of model vs. view space. The gradient texture approach highlighted the power of artist driven control over pure math. Finally, integrating everything under a single ImGui UI and porting seamlessly to WebGL strengthened my end to end graphics pipeline expertise.
