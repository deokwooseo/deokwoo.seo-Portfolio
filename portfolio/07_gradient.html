Project Gradient Noise

3D Graphics

Project Overview

Implemented a 3D gradient noise generator on the GPU to create various noise patterns and visualize them as textures and terrain surfaces.

Tasks Completed

  • Wrote a vertex and fragment shader to produce improved Perlin gradient noise in 3D
  • Created an offscreen framebuffer with an R32F texture to store noise values
  • Added support for plain gradient noise fractal sum turbulence marble and wood patterns
  • Built a simple shader to display the noise slice as a textured quad and another to displace a plane mesh into a terrain surface

Reflection

When running the web release build I saw an Uncaught 1356536 error that I had not encountered before. After debugging I discovered the frame buffer code tried to access colorTexture through a private method. Moving the Texture& ColorTexture() method into public fixed the error. This taught me to check access modifiers carefully when exposing GPU resources.
