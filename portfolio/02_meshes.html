Project Procedural Geometric Modeling

3D Graphics

Project Overview

The project focuses on procedurally generating a variety of 3D geometric models using OpenGL and GLSL as part of my CS250 Computer Graphics course at DigiPen. The objective was to create dynamic mesh generation routines for shapes such as planes, cubes, spheres, cylinders, cones, and toruses, and to correctly construct their vertex and index buffers for real-time rendering.

Tasks Completed

  • Procedural Mesh Generation: Developed algorithms to generate 3D shapes using parametric equations and trigonometric functions, ensuring that the resulting meshes were watertight and correctly indexed.
  • Vertex Layout and Buffer Mapping: Designed custom vertex structures containing position, normal, and UV attributes, and correctly mapped them into OpenGL buffer layouts with attribute descriptors for efficient rendering.

Reflection

One of the primary challenges was managing the intricacies of mesh topology and ensuring consistent index buffer generation across different shapes. Debugging issues related to vertex ordering and winding helped me gain a deeper understanding of both the rendering pipeline and efficient buffer management. Overall, this project reinforced the importance of modular design and thorough testing in graphics programming.
